# MSCS532 Assignment 1 - Insertion Sort

This project implements the Insertion Sort algorithm in Python to sort a list of integers in monotonically decreasing order (largest to smallest).

## Requirements

* Python 
* Visual Studio Code
* Git and GitHub

## How to Run

1. Open the project folder in Visual Studio Code.
2. Open the terminal.
3. Run the following command:

```bash
python insertion_sort.py
```

## Example

**Input:**

```text
[10, 4, 25, 8, 2, 15]
```

**Output:**

```text
[25, 15, 10, 8, 4, 2]
```

## Algorithm Details

The program uses Insertion Sort to arrange numbers in decreasing order. It compares each value with the values before it and moves smaller values to the right.

* Time Complexity: O(n²)
* Space Complexity: O(1)
